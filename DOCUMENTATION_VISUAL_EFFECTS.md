# Эффект Всплеска Воды (Лейка)

## 🎨 Что Добавлено

### Визуальный Эффект:
- **8 частиц всплеска** (ParticleTypes.SPLASH) - разлетаются в стороны
- **3 капли воды** (ParticleTypes.FALLING_WATER) - падают вниз
- Появляется **на каждой политой грядке**
- Работает со всеми паттернами (SINGLE, THREE, FIVE, GRID_3X3, GRID_5X5)

---

## 🎯 Как Это Работает

### При Поливе Одной Грядки (SINGLE):
```
Клик → Грядка поливается → 💦 Всплеск воды!

Эффект:
- 8 брызг воды разлетаются в стороны
- 3 капли падают вниз
- Звук полива (BOTTLE_FILL)
```

### При Поливе Нескольких Грядок (THREE):
```
Клик → 3 грядки поливаются → 💦💦💦 Всплески на каждой!

Эффект:
- Каждая грядка = отдельный всплеск
- Синхронная анимация
- Красивый визуальный фидбек
```

### При Поливе Большого Паттерна (GRID_3X3):
```
Клик → 9 грядок поливаются → 💦💦💦
                              💦💦💦
                              💦💦💦

Эффект:
- 9 одновременных всплесков
- Эффектная "волна" полива
- Моментальная обратная связь
```

## 🔧 Технические Детали

### Метод Спавна Частиц

```java
private void spawnWaterSplashParticles(ServerLevel level, BlockPos pos) {
    // Позиция чуть выше грядки
    double x = pos.getX() + 0.5;
    double y = pos.getY() + 1.0;
    double z = pos.getZ() + 0.5;
    
    // Спавним несколько частиц всплеска
    for (int i = 0; i < 8; i++) {
        double offsetX = (level.random.nextDouble() - 0.5) * 0.6;
        double offsetZ = (level.random.nextDouble() - 0.5) * 0.6;
        double velocityY = level.random.nextDouble() * 0.2 + 0.1;
        
        level.sendParticles(
            ParticleTypes.SPLASH,
            x + offsetX,
            y,
            z + offsetZ,
            1,
            0, velocityY, 0,
            0.1
        );
    }
    
    // Добавляем немного капель воды
    for (int i = 0; i < 3; i++) {
        double offsetX = (level.random.nextDouble() - 0.5) * 0.4;
        double offsetZ = (level.random.nextDouble() - 0.5) * 0.4;
        
        level.sendParticles(
            ParticleTypes.FALLING_WATER,
            x + offsetX,
            y + 0.3,
            z + offsetZ,
            1,
            0, 0, 0,
            0
        );
    }
}
```

### Вызов Эффекта

## 📊 Параметры Частиц

### ParticleTypes.SPLASH (Всплеск)

| Параметр | Значение | Описание |
|----------|----------|----------|
| Количество | 8 | Частиц на грядку |
| Позиция Y | pos.Y + 1.0 | Чуть выше грядки |
| Разброс X/Z | ±0.3 блока | Случайное смещение |
| Скорость Y | 0.1 - 0.3 | Подбрасывает вверх |
| Spread | 0.1 | Разброс траектории |

| Параметр | Значение | Описание |
|----------|----------|----------|
| Количество | 3 | Капель на грядку |
| Позиция Y | pos.Y + 1.3 | Выше всплесков |
| Разброс X/Z | ±0.2 блока | Меньше разброс |
| Скорость | 0 | Падают под гравитацией |

# Эффект Вспашки Земли (Мотыга)

## 🎨 Что Добавлено

### Визуальный Эффект:
- **8 желто-оранжевых частиц пыли** (DustParticleOptions, цвет #ebab22) - разлетаются в стороны, имитируя пыль земли

## 🔧 Технические Детали

### Метод Спавна Частиц

```java
private void spawnTillingParticles(ServerLevel level, BlockPos pos) {
    // Позиция чуть выше блока
    double x = pos.getX() + 0.5;
    double y = pos.getY() + 1.0;
    double z = pos.getZ() + 0.5;
    
    // Желто-оранжевый цвет для частиц земли (RGB: #ebab22 -> 235, 171, 34)
    Vector3f earthColor = new Vector3f(0.922f, 0.671f, 0.133f);
    DustParticleOptions dustOptions = new DustParticleOptions(earthColor, 1.0f);
    
    // Спавним желто-оранжевые частицы редстоуна (имитация пыли земли)
    for (int i = 0; i < 8; i++) {
        double offsetX = (level.random.nextDouble() - 0.5) * 0.6;
        double offsetZ = (level.random.nextDouble() - 0.5) * 0.6;
        double velocityY = level.random.nextDouble() * 0.2 + 0.1;
        
        level.sendParticles(
            dustOptions,
            x + offsetX,
            y,
            z + offsetZ,
            1,
            0, velocityY, 0,
            0.1
        );
    }
    
    // Добавляем частицы пепла (как облачко земли)
    for (int i = 0; i < 3; i++) {
        double offsetX = (level.random.nextDouble() - 0.5) * 0.4;
        double offsetZ = (level.random.nextDouble() - 0.5) * 0.4;
        
        level.sendParticles(
            ParticleTypes.ASH,
            x + offsetX,
            y + 0.1,
            z + offsetZ,
            1,
            0, 0.05, 0,
            0.02
        );
    }
}
```

### Вызов Эффекта

```java
// При вспашке земли
if (state.is(ModBlocks.DIRT.get())) {
    level.setBlock(pos, ModBlocks.FARMLAND.get().defaultBlockState(), 3);
    level.playSound(null, pos, SoundEvents.HOE_TILL, SoundSource.BLOCKS, 1.0F, 1.0F);
    
    // Добавляем эффект вспашки земли
    if (level instanceof ServerLevel serverLevel) {
        spawnTillingParticles(serverLevel, pos);
    }
}

// При обработке грядки
else if (state.is(ModBlocks.FARMLAND.get())) {
    level.playSound(null, pos, SoundEvents.HOE_TILL, SoundSource.BLOCKS, 1.0F, 1.0F);
    
    // Добавляем эффект вспашки земли
    if (level instanceof ServerLevel serverLevel) {
        spawnTillingParticles(serverLevel, pos);
    }
}
```

## 🚀 Оптимизация

### Общие Характеристики:
- ✅ **Серверная генерация** - частицы спавнятся на сервере
- ✅ **Автоматическая синхронизация** - все игроки видят эффекты
- ✅ **Временные частицы** - исчезают через ~1 секунду
- ✅ **Случайность** - каждый эффект уникален
- ✅ **Оптимизация** - оптимальное количество частиц

### Нагрузка по Паттернам:

| Паттерн | Частиц | Нагрузка | Подходит для |
|---------|--------|----------|--------------|
| SINGLE | 11 | Минимальная | Все компьютеры |
| THREE | 33 | Низкая | Все компьютеры |
| FIVE | 55 | Низкая | Все компьютеры |
| GRID_3X3 | 99 | Средняя | Средние+ компьютеры |
| GRID_5X5 | 275 | Умеренная | Хорошие компьютеры |

### Оптимизация:
- ✅ Частицы **не постоянные** (только при использовании)
- ✅ **Короткое время жизни** (~1 секунда)
- ✅ **Батчинг на сервере** (эффективная передача)
- ✅ **Клиентский рендеринг** (нагрузка на GPU, не CPU)
